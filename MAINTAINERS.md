## Becoming a Maintainer

Contributors who submit sustained, high-quality work and demonstrate good
judgment in code review are invited to join the maintainer team. Nominations
are opened by an existing maintainer and require unanimous approval from the
current maintainers.

## Stepping Down

Maintainers who can no longer dedicate time are encouraged to hand off
responsibility explicitly. Update this file and open a pull request listing the
new assignments.
