# Contributor Covenant Code of Conduct

This project adopts the Contributor Covenant, version 2.1.

The full text is available at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Scope

This code applies to all project spaces: the code repository, issue tracker,
pull requests, discussion forums, mailing lists, chat channels, and any event
organized under the project's name.

## Reporting

To report a concern, contact the maintainers listed in
[MAINTAINERS.md](MAINTAINERS.md). All reports will be handled in confidence.
Maintainers who violate the code of conduct in good faith may be temporarily or
permanently removed from the project.
